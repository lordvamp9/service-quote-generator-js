# ClearQuote - Service Quote Calculator

A modern, responsive web application designed to provide instant service estimates for digital agencies and freelancers.

## Features

- **Interactive Quote Calculator**: 3-step process to estimate project costs.
- **Dynamic Pricing**: Real-time updates based on service tier and add-ons.
- **Responsive Design**: Fully optimized for mobile, tablet, and desktop.
- **Professional UI**: Clean, modern interface built with Bootstrap 5 and custom CSS.
- **Contact Integration**: Seamless flow from quote generation to contact form.

## Technologies Used

- HTML5
- CSS3 (Custom + Bootstrap 5)
- JavaScript (Vanilla)
- Font Awesome

## Setup

1. Clone the repository.
2. Open `index.html` in your browser.
3. No build process required.

## Project Structure

- `index.html`: Main landing page and calculator.
- `about.html`: Company information.
- `services.html`: Detailed service breakdown.
- `contact.html`: Contact form.
- `assets/css/`: Stylesheets.
- `assets/js/`: JavaScript logic.
